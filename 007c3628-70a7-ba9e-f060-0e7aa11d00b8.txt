　　过万武者，惊愣着。
<br/>　　周遭，山壁嶙峋，似乎是在某个巨大的山洞里。
<br/>　　而四周的邪修，密密麻麻，数量多得惊人。
<br/>　　判断是否邪修，并非是根据那身黑袍着装，而是根据他们身上那股让人心悸的阴寒气息。
<br/>　　邪修所修之邪道，与正常武者的正常气息有极大的区别。
<br/>　　若是邪修没有可以隐藏自己的气息，正常武者一眼便能认出。
<br/>　　自然的，山洞之内，环伺四周的这为数惊人的邪修，让得一众武者，惊愣在原地。
<br/>　　不过，让人奇怪的是，周遭邪修，同样愣了愣，显然对于忽然出现的一众武者感到惊讶。
<br/>　　“怎…怎么…”
<br/>　　双方，同时惊愣着。
<br/>　　偌大个山洞之内，竟一时陷入了沉寂。
<br/>　　这沉寂，并不显得沉重，反倒是在双方大眼瞪小眼的情况下，显得有些滑稽。
<br/>　　忽然，一个武者率先惊呼出言，打破了沉寂。
<br/>　　“火…火牙宗主？你…你也是邪修？”这个武者，声音中带着不可置信。
<br/>　　不错，周遭邪修中，火牙宗主赫然在内，包括火牙宗的长老们，以及一些弟子。
<br/>　　哗…
<br/>　　沉寂的气氛，忽然一瞬间爆发。
<br/>　　所有人都从惊愣中反应了过来。
<br/>　　“邪修，是邪修，这里是邪修集聚的分部。”过万武者中，一个明显实力极强的老者怒吼一声。
<br/>　　一声之下，一众武者，霎时露出了警惕之色，也做好了战斗的准备。
<br/>　　邪修二字，在中域之内无论何时都是让人闻之色变的存在。
<br/>　　那残忍而嗜血的手段，那阴寒而诡异的气息，无不让人惊惧。
<br/>　　另一边，一众邪修亦在同一时间反应了过来。
<br/>　　火牙宗主一声冷喝，“包围他们。”
<br/>　　火牙宗主，没有解释半句，而是瞬间下了命令。
<br/>　　这等情况，根本不需要解释，解释也无用。
<br/>　　一众武者，发现了火牙宗便是邪修分部，已是事实。
<br/>　　不过，火牙宗主此刻的脸色，难看到极点。
<br/>　　“该死，你们怎么会出现在这里？”火牙宗主冷眼扫视着一众武者。
<br/>　　“我们怎会出现在这里？”几乎也是同一时间，各大势力的一些强者惊疑说着。
<br/>　　双方，都处于惊疑。
<br/>　　周遭气氛，变得莫名了起来。
<br/>　　当然，双方都对峙着，并无半分放松。
<br/>　　这时，一众武者中，一道单薄的身影，缓缓走出。
<br/>　　“火牙宗主，可还认得我。”身影，冷笑一声。
<br/>　　正是廖灵儿。
<br/>　　“你是？”火牙宗主，双眼一眯。
<br/>　　“呵呵。”廖灵儿莫名一笑，“也是，火牙宗主可是大人物。”
<br/>　　“在火牙地域内，是有数的强者，是人人敬仰的火牙宗宗主。”
<br/>　　“在邪君府内，亦是身居高位，统领数百地域的邪修分部。”
<br/>　　“你这等大人物，记不得我这种小人物也是应该。”
<br/>　　廖灵儿，笑着，却也带着冰冷的齑粉。
<br/>　　“你到底是何人？”火牙宗主的脸色，已然发黑。
<br/>　　他不知道发生了什么，但必然与面前这个粗布麻衣，看似普通的女子有关。
<br/>　　他火牙宗，暗中作为邪修分部多年，一直无人发现。
<br/>　　他一直是高高在上的火牙宗主，火牙宗也一直是人人敬仰的大势力。
<br/>　　但今日，一个不慎，怕是要酿成极大的后果。
<br/>　　他火牙宗这些年的心血，也可能功亏一篑。
<br/>　　自然的，他的脸色极其难看。
<br/>　　廖灵儿冷笑一声，“火牙宗主记不得我，那可还记得十年前来你火牙宗参加火牙盛事的一对兄妹？”
<br/>　　“嗯？”火牙宗主若有所思，似是想起了什么。
<br/>　　“原来是你。”火牙宗主蓦地脸色一冷，“十年前那只小老鼠。”
<br/>　　“看来是想起来了。”廖灵儿嘴角咧过一道讥讽冷笑。
<br/>　　“十年前，我与我哥来此参加盛事，只为得些奖励，好修习武道。”
<br/>　　“我哥一举斩获盛事第一，我至今还记得火牙宗主当初亲自为我哥授予奖励，收为门下弟子时的威严。”
<br/>　　“谁曾想，堂堂火牙宗主，竟是邪修长老。”
<br/>　　“又谁能想到，背地里的火牙宗主，冷笑之声阴冷如斯。”
<br/>　　“我哥不愿与你等同流合污，你便要将他击杀，甚至炼成傀儡。”
<br/>　　“呵。”廖灵儿冷呵一声，“我同样至今还记得，我哥前一日还视你为恩师，后一日便在你那副狰狞嘴脸下，变得不可置信，狼狈至极。”
<br/>　　“可再怎么狼狈，他还是护着我安然离了你这表面光鲜，暗地龌蹉的阴暗之地。”
<br/>　　“十年后的今日，我便回来了。”
<br/>　　廖灵儿的脸色，冰冷到极点。
<br/>　　“然后呢？”火牙宗主不屑一笑，“十年前，那个废物明明掌控着世间罕见的空间一道，却非要自寻死路。”
<br/>　　“当然，我也不妨告诉你，若非那个废物当日要护你周全，他根本不可能殒命。”
<br/>　　“空间一道，是传奇强者才可能掌控的手段；传奇强者以下，任何人都难以奈何。”
<br/>　　“那个废物一心要逃，我也奈何不得；偏偏他还带着你这拖油瓶，便也只能拿他的命，来换你的命。”
<br/>　　火牙宗主此刻的脸色，异常狰狞。
<br/>　　而廖灵儿，原本脸上的冰冷和得意，一瞬间凝固。
<br/>　　“傻瓜。”火牙宗主桀桀冷笑，“与其说是我杀了你哥，倒不如说是你害死了你哥。”
<br/>　　“什么…”廖灵儿身躯一颤，眼眸霎时陷入失神。
<br/>　　“哼。”恰在此时，一声冷哼，从廖灵儿心头响起。
<br/>　　原本失神的廖灵儿，霎时恢复了清明。
<br/>　　“嗯？”火牙宗主眼中，闪过一丝异色，“这么快就能醒来？看来若些本事。”
<br/>　　廖灵儿一阵心有余悸，她知晓，并非是她这么快醒来。
<br/>　　而是刚才那一声莫名响起的冷哼，让她恢复了清明。
<br/>　　火牙宗主，不愧是极强的邪修，寥寥数语，差点攻破了廖灵儿的心神。
<br/>　　若廖灵儿刚才没有及时醒来，怕是会心神受损，即便火牙宗主不出手，她也会死在自己手上。
<br/>　　“邪修，果真是手段诡异，又狡猾过人。”廖灵儿冷哼一声。
<br/>　　“想扰我心神？是你们杀了我哥，今日，我是来报仇的。”
<br/>　　“报仇？”火牙宗主不屑一笑，“凭什么？”
