        &lt;h3&gt;第五十章  陆玉调动&lt;/h3&gt;
<br/>
<br/>        陆玉已经启程去甸城了，她对甸城充满了向往。
<br/>
<br/>        此次去甸城，她是和秦少卿在一起工作的，因为年龄的关系，陆玉去甸城也是副手，但是有陆家的这层关系，陆玉也不可能会吃亏的，况且秦家和陆家的关系就非比寻常。
<br/>
<br/>        陆战君亲自送女儿去甸城上班。
<br/>
<br/>        虽然甸城地处偏远，交通不算是很好，但是还是有一条要道直通甸城，陆战君带了三辆车，车的规格也是相当的高。
<br/>
<br/>        陆玉此次去甸城他是支持和鼓励，甸城这几年发展也很快，那边的产业虽然没有投资，实际上他利用手中的资源比投资还要有用，谷家和秦家也越来越大了，听口气似乎想要做一笔大生意。
<br/>
<br/>        放陆玉过去，第一是锻炼陆玉，第二是监视谷家和秦家，陆家需要发展，也是需要资源的支撑，这几年各大家族随着经济开放浪潮的兴起，随着时代发展的变迁，在各种产业上都是独占鳌头，每个家族手中都有强大的经济作为后盾。
<br/>
<br/>        人民生活日益富裕，他们的生活也需要富裕，不但要富裕，还要掌握一部分经济命脉，握在手中的东西才能是自己的。
<br/>
<br/>        甸城地处偏远，有利有弊，利，上面无法顾及，各种产业暴利，弊，也是因为太远不好掌控，秦寿和他儿子是他一步一步的提起来的，他，就是秦寿上面的一把伞，为秦家和谷家遮风挡雨。
<br/>
<br/>        “玉儿，此次去甸城工作，一定要和秦家、谷家搞好关系，我们几家是关系好，那边就是他们的地盘，还有，两家的动向你也得及时掌握，向我汇报。”
<br/>
<br/>        “好的，爸，你放心，我一定和他们搞好关系，你也知道我和正娟和少卿的关系都很好呢。”
<br/>
<br/>        “谷正娟和秦少卿年纪轻，不足为虑，你要注意的是谷老大和秦寿，这就是两只老狐狸，他们手中掌握着很多边缘产业，你必须给我盯紧了。我们陆家的这些年的经济支出都是靠他们来支出，而，我们陆家要发展，必须得有强大的经济作为后盾。你明白我的意思吗？”
<br/>
<br/>        “明白的。”陆玉从小都是家里宠着长大的，从她记事起，吃的用的都是最好的，在农村孩子饥肠寡肚的时候，她却在山珍海味，燕窝鲍鱼。
<br/>
<br/>        陆家看起来就是兢兢业业的工作着，他们被敬业的光环笼罩着，他们努力的工作，为了大众的利益而奋斗着。
<br/>
<br/>        陆战君点点头，陆家他为大，陆家的担子他要一肩挑，俩个孩子也都是很聪明能干，上学和工作都是顺风顺水，虽然有陆家的光环罩着，但是他们也是足够努力，一个个的学业有成，女儿虽然养的有些骄纵跋扈，但是不可否认，她也是一个人才，而且做事情比儿子要大胆很多。
<br/>
<br/>        陆玉与谷家、秦家的关系比陆宇霆要熟悉很多，接触的事情也多，放陆玉过去是最合适不过的，呆上四五年，回来也有晋升的机会。
<br/>
<br/>        陆战君并没有想到的是，他的女儿心思并不在事业上，而是在美男秦少卿的身上。
<br/>
<br/>        陆战君看着车外飞驰而过的景色，陷入回忆中。
<br/>
<br/>        他认识秦寿之时，秦寿还是一个小的角色，秦寿年轻能干，做事情活套，每次来北京都会特意来看他，顺便给他带一些东西，慢慢的俩个人关系也就越来越近了。
<br/>
<br/>        秦寿没有人脉，他就给秦寿找点人脉，秦寿感激他，给他送了不少东西，他不要，秦寿说：“陆总，我虽然在山里，但是这边的情况可是看得清清楚楚，别人家那可是有强大的经济在支撑着啊，一切的发展都离不开经济，要不然那些个老总一定要让自己的孩子去发展经济?这个是趋势啊。”
<br/>
<br/>        陆战君当然明白这个道理，只是他们目前确实没有什么好的产业可以拿到手。
<br/>
<br/>        “我们甸城，有个谷老大，甸城的大部分产业基本都是他的，而他是我的同学，我们的关系极好。陆总，您入一股如何？”
<br/>
<br/>        “入股？”
<br/>
<br/>        “干股。”
<br/>
<br/>        这就是谷老大和秦寿在甸城独占鳌头的原因。
<br/>
<br/>        记得有一年，秦寿带了一皮箱子花花绿绿的外币回来，当时他没有在家，放别人哪里他不放心，想起他的好友张沛尧，然后让秦寿绕道西城，将货放在张沛尧家里。
<br/>
<br/>        他不确定张沛尧有没有看到皮箱里面的东西，曾经旁敲侧击的问过，但是张沛尧好像是没有发现，他的心里始终是一根刺，陆玉想要嫁给张函，他明知道张函不爱陆玉，但是他利用关系将张沛尧拉回北京，以此让张家同意陆玉嫁给张函。
<br/>
<br/>        张家如今和他是一条绳上的蚂蚱了。
<br/>
<br/>        陆玉看着父亲陷入沉思，便也没有打扰父亲，也默默的回忆起走时的情景。
<br/>
<br/>        张函帮着她整理行李，其实她也没有多少东西要拿的，过去都会有的。
<br/>
<br/>        “陆玉，过去就好好工作，争取过几年回来能更上一层楼。”张函这几年已经褪去了青年时代的狂傲不羁，显得沉稳而凌厉，好看的眉眼一如既往，更是增添了一种成熟男人的美。
<br/>
<br/>        只是这美是可望不可即的，和她始终是隔着千山万水。
<br/>
<br/>        她在他的眼里从来都没有发现过温柔，即使俩个人在一起看着很恩爱的样子，张函的眼里始终是没有温度的。
<br/>
<br/>        当她看到秦少秦看着闫芈温柔似水的眼神，看着那种宠爱，那种呵护，她怦然心动。
<br/>
<br/>        原来，这个世界上还有那样温柔的男人！
<br/>
<br/>        想起秦少卿，陆玉的心里无端的柔软起来，划过一丝奇异的暖流，虽然秦少卿的温柔不是对她的，但是她希望不久的将来，秦少秦会那么对待她的。
<br/>
<br/>        “我知道了，没发现，你像一个老婆子一样唠唠叨叨啊，比我妈还让人厌烦。”陆玉有点不耐烦。
<br/>
<br/>        “我这不是担心你嘛，一个人去那么远的地方。”
<br/>
<br/>        “有什么好担心的，那边两大家族都是我们自己的人，再说，我过去是工作的，又不是参加战斗。父亲这一次也会和我一起过去的。”
<br/>
<br/>        “父亲也过去？专门送你啊？你真正是大家的心肝宝贝。”
<br/>
<br/>        “父亲和谷家，秦家相交已久，我们陆家的产业就在那边，只是有他们两家打理着，父亲这一次也是过去看看，而我，过去的大部分原因就是看着他们。”陆玉毫不隐瞒张函。
<br/>
<br/>        “我都不知道那边还有产业呢，难怪咱陆家发展的势头愈来愈好了。”
<br/>
<br/>        “你不知道的事情多着呢。”
<br/>
<br/>        “那，那边的产业都有些什么？”
<br/>
<br/>        “我也不知道，据说涉及的很多，方方面面也都有接触，那边的形式你是知道的。”陆玉看了一眼张函。
<br/>
<br/>        “你以后有些闲事情少管，要不是你是我陆玉的男人，怕是你也不能落个好下场的。”陆玉意有所指。
<br/>
<br/>        张函听得心里一惊。
<br/>
<br/>        “那是，我要不是你陆玉，怎么也不会有今天的。”
<br/>
<br/>        “知道就好，你也好好工作，将来我们齐头并进，也不失为一段佳话。”
<br/>
<br/>        “好!”
<br/>
<br/>        陆玉叹了口气，这几年张函确实是对她不错，可是她毕竟是一个女人，一个有着正常需求的女人。
