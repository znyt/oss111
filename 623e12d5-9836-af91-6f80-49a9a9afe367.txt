　　第376章
<br/>　　这是一双拿刀的手啊，马背才是你的舞台，如此捧着一张琵琶，是不是走错了片场。
<br/>　　但现在樊梨花情绪有些不对，程大雷也只能捧着聊。
<br/>　　“学会了么？”程大雷轻声问。
<br/>　　“学会了一点。”
<br/>　　“啊！这真是难以想象，不知道我可有耳福欣赏。”
<br/>　　樊梨花白了程大雷一眼，轻轻颦眉，手指拨弄琵琶。程大雷装出专注模样，那声音……程大雷就感觉樊梨花用刀砍掉别人脑袋时，血从腔子里流出的声音也比这悦耳些。
<br/>　　“怎么样？”樊梨花认真问。
<br/>　　“啧！”程大雷赞了一声：“此曲只应天上有，人间更得几回闻，我真不知该用什么词形容。”
<br/>　　程大雷一点没有说谎，是难听得不知道怎么形容。
<br/>　　樊梨花很显然是信了，脸上有了些笑容。自从出事后，程大雷还是很少见到樊梨花笑，今日一见，顿如梨花初绽，程大雷一时看得有些痴了。
<br/>　　“琴乐只是小道，你怎么会喜欢这些？”程大雷赶紧抓住机会问。
<br/>　　“琴乃心动之音，乐通自然之理，怎么能说是小道呢。你也懂这些？”
<br/>　　程大雷一听就知道，这话是苏樱和柳芷熏陶的。苏樱是商人之女，柳芷出身武将世家，其实家族都没什么文化，所以对有文化的人比较崇拜，而她们对这些，其实也不怎么懂。山寨最懂这些的是李婉儿，反而不太在意。
<br/>　　“我不太懂的。”程大雷老实道。
<br/>　　“我就知道你不懂。”
<br/>　　程大雷无语，这话怎么听着有种鄙视的意思。我是没文化的人么，你知不知道我在京城有多少拥泵。
<br/>　　程大雷陪着樊梨花说些闲话，自从樊梨花出事之后，他便经常过来，想要开导对方。二人能说的话，也说了不少，到现在，每天说得不过是一些闲事而已。程大雷就聊些诺顿王朝的事，还有帝国最近的状况。
<br/>　　可看樊梨花的意思，也不如何关心，有一句没一句的应着。但在程大雷要走的时候，眼神中又有些不舍之意。
<br/>　　至于樊梨花心里想什么，程大雷是不懂的。
<br/>　　女儿心情总是诗，而诗属于高级文学，总是难懂的。
<br/>　　其实樊梨花也没有那么难以揣测，崔弄海的事情，的确在她心里留下梦魇。也不怪她胆小，任何一人遭遇那样的魔鬼，怕都会留下心理创伤，何况樊梨花还是个女人。
<br/>　　入夜时分，樊梨花都会在噩梦中惊醒，然后不敢再入睡。程大雷也派了丫鬟照顾她，可樊梨花却也不能再信任任何人，半夜总是将丫鬟赶出去。
<br/>　　唯一能令她安心的，也就是程大雷了。可她一个女儿家，总不能要求程大雷陪着自己睡觉，草原姑娘，也是要脸的。
<br/>　　每日如此，身体渐渐消瘦。而且每过去一年，就增一岁，青春渐逝，终身无托，也难怪她不开心。
<br/>　　女儿心思不能对程大雷谈，其他人又信不过，偏偏樊梨花还是个要强的，只能强撑着，越撑越颓。
<br/>　　程大雷当真有些束手无策，樊梨花是道难题，不知该如何下手解。他眨了眨眼睛：“要不饮些酒？”
<br/>　　“嗯，好吧。”樊梨花对吃酒无兴趣，只是想多留程大雷片刻。
<br/>　　“我去让人送酒过来。”程大雷准备起身。
<br/>　　“不用，我床底下有。”
<br/>　　樊梨花的床底下果然摆着一坛坛酒，倒是吓了程大雷一跳。一腔心思无人诉说，她也就只能给了酒。
<br/>　　二人拍开一坛，分杯而饮，程大雷倒是能喝些，却也没多大兴趣。推杯换盏，不知过了多时，最后醉眼惺忪，只见明月挂在中天，程大雷两眼一黑，醉了过去。
<br/>　　此时天明，程大雷悠悠醒转，发现自己睡在樊梨花床上，扑鼻幽香。他吓了一跳，回过神来，见樊梨花趴在桌上，此刻还熟睡着。
<br/>　　程大雷轻松口气，从床上下来，将樊梨花抱在床上。看着她睡梦中轻轻眨动睫毛的模样，忍不住在她脸颊上拍了拍。
<br/>　　之后程大雷踏出屋子，走到自己院子，柳芷和小桃看程大雷眼神有些不对了。
<br/>　　程大雷也懒得解释，解释起来似乎也没那么清白。苏樱帮他洗漱，更衣，程大雷直着双臂，总感觉气氛有些不对。
<br/>　　他隐隐觉得苏樱有话要说，可苏樱一直不说，程大雷也不知怎么开口。
<br/>　　最后，苏樱帮他系上扣子，拍了拍他的胸口，道：“人家姑娘大了，你总要给个交待，姑娘和你不一样，拖不起的。”
<br/>　　程大雷点点头，示意自己明白，这是件心病，总是要除根的。
<br/>　　一夜间，秦蛮已经将夷族一批人接了过来。据秦蛮所说，这些人像一辈子没吃过猪肉一样，馋得要命。而且饮食等生活习惯也和这里不同，把他们丢在住处，任他们自己收拾去吧。
<br/>　　不过，这波人的首领要见一见程大雷，大概是表示感谢之类的。
<br/>　　他们是寄人篱下，以程大雷现在的身份，自然不需要特意等他们，而是他们等程大雷有时间。
<br/>　　程大雷也没有摆谱，安排在将军府的偏厅见面。他令丫鬟上了茶，过不多时，徐神机、凯恩还有一白袍女子进来。
<br/>　　这女子就是凯恩的首领，身份是新教的圣女，教义中是诸神在人间的代言人，名叫奥黛琳娜。
<br/>　　据秦蛮说，这女子长得很漂亮，程大雷一见，果然名不虚传。奥戴琳娜不仅有夷族修长的身体，雪白的皮肤，身上隐隐有一种神性，这大概是男人最想征服的。当然，也可以称之为糟蹋，男人骨子里都有破坏欲的。
<br/>　　不过，程大雷倒是没有什么心思，一个樊梨花他还不知道怎么办呢。
<br/>　　奥黛琳娜走入后，按照他们的礼节向程大雷行了一礼，道：“感谢程当家，愿诸神的祝福永远庇佑你。”
<br/>　　“好说，好说。”程大雷摆摆手：“坐吧。”
<br/>　　奥黛琳娜落座，凯恩站在她身后，程大雷撇撇嘴，派头还挺大。
<br/>　　“到了这里，你们就可以放心，没有人能伤害你们，你们能安心过自己的生活。”对美女，程大雷总会客气些，也不介意显摆一下自己的实力。
<br/>　　“感谢程当家的照顾，能否请程大雷护送我们前往帝国的首都长安，面见帝国的皇帝。”
<br/>(本章完)
