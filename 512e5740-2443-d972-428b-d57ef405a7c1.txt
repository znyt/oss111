　　没有了妖龙精骨，此时叶无伤的实力，也就是人仙级别罢了，再加上元气大伤，自然远远对叶尘构不成威胁。
<br/>　　“不！我怎么可能会输！这不可能！不可能！”
<br/>　　叶无伤疯狂的大吼大叫，似乎已经丧失理智。
<br/>　　“前世今生，两世之仇，一切都结束吧！”
<br/>　　叶尘低声喃喃自语了一句之后，眼眸中闪过一抹厉色，然后抬起手掌，便要将叶无伤直接灭杀。
<br/>　　然而就在这时，
<br/>　　嗖！
<br/>　　叶尘在叶无伤之间，陡然出现了一道人影，同时一道暴喝声响起，
<br/>　　“小尘！你不能杀他！他毕竟是你的哥哥！”
<br/>　　这个突然出现的人影，自然正是叶天歌。
<br/>　　叶尘不由得冷冷一笑，
<br/>　　“哥哥？叶天歌，看来你到现在，都还没有搞清楚状况，我和你都没有任何关系，你觉得我会认他？”
<br/>　　叶天歌顿时一滞，脸色惨白无比，
<br/>　　“不管你认不认，你们两人的血缘关系是既定事实，你永远改变不了！”
<br/>　　“那又如何？害我亲人，杀我朋友，管他是谁，我必杀之！”
<br/>　　叶尘依旧一脸平淡，缓缓开口道。
<br/>　　“你！”
<br/>　　叶天歌见叶尘丝毫不为所动，顿时有些束手无策，最终一咬牙道：
<br/>　　“好！你如果真要杀，那就杀我好了！我替他死！”
<br/>　　不料，叶尘听到这话，顿时哈哈大笑，
<br/>　　“叶天歌，你好像忘了一件事情吧！你的命，是我救活的，所以你无权做主！你若真想死，那就死一边去，叶无伤的命，你抵不了！”
<br/>　　“你……”
<br/>　　叶天歌顿时彻底无言以对。
<br/>　　这时，下方苏岚的声音陡然响起，
<br/>　　“小尘！你不可以和你爸爸这么说话！”
<br/>　　听到苏岚发话，叶尘不由得暗暗一叹，
<br/>　　叶天歌他可以不理会，可是苏岚他却不能置之不理。
<br/>　　“岚姨，我和叶天歌之间的恩怨，请您就不要管了！”
<br/>　　“至于叶无伤，他害死了姜长老和尹庄主，打伤蔓姐，又险些杀了你，我岂能不杀？”
<br/>　　“这个……”
<br/>　　苏岚听到这话，也顿时无言以对。
<br/>　　这几个月来，尹百川守在天海大学，保护苏蔓的安危，不知道帮她躲过了多少次明枪暗箭。
<br/>　　苏岚对他一直心存感激！
<br/>　　如今尹百川，死在了叶无伤的手上，让叶尘放弃为他报仇，苏岚自问做不到……
<br/>　　而逍遥山庄的众弟子，此时也群情激愤，纷纷高呼着，要为尹百川和姜长老报仇。
<br/>　　这时，叶无伤忽然疯狂大笑起来，
<br/>　　“小野种！成王败寇！今日我败给了你，我认了，可是我不需要你的可怜！”
<br/>　　“我劝你最好赶快杀了我！否则只要我还有一口气在，我发誓！我会杀光你所有的亲人朋友！桀桀桀！”
<br/>　　听到这话，叶尘顿时脸色猛然一寒，身上杀气升腾，
<br/>　　“好！我这就成全你！”
<br/>　　声音落下的同时，叶尘身体一晃，已经绕过空中的叶天歌，来到叶无伤的跟前，然后狠狠一掌拍落了下去！
<br/>　　彭！
<br/>　　叶无伤此时已经毫无任何反抗之力，瞬间便被叶尘这道强悍无比的掌力，直接轰成一团血雾！
<br/>　　“不！！”
<br/>　　后方的叶天歌见状，顿时大吼了一声，可惜已经无力回天。
<br/>　　叶天歌两眼瞪得老大，呆立了片刻之后，
<br/>　　“噗！”
<br/>　　忽然口中狂喷一口鲜血，继而一头栽落了下去。
<br/>　　“天歌！”
<br/>　　下方的苏岚，看到这一幕，当即快步上前，将叶天歌从地上扶起。
<br/>　　……
<br/>　　也不知为何，将叶无伤轰杀之后，叶尘心里没有喜悦，竟然有些说不出的难受。
<br/>　　当然，这种难受，并非是那种后悔的难受，因为他可以百分百确定，他一点也不后悔杀死叶无伤。
<br/>　　甚至，即便是给他一百次选择的机会，他还是会毫不犹豫的这么做。
<br/>　　可是此刻，他的心里，就是很难受，很压抑，很不舒服！
<br/>　　并且还说不上来什么原因！
<br/>　　“飞烟，接下来的事情，你来处理吧！”
<br/>　　叶尘忽然感觉整个人很疲倦，想好好休息一下，于是冲着楚飞烟吩咐了一声，然后便朝黑森林深处，缓缓走去……
<br/>　　“老师，你去哪儿？”
<br/>　　小林舞依想要追过去，却被楚飞烟一把拉住，
<br/>　　“别去！你让他一个人，好好静一静吧！唉！”
<br/>　　楚飞烟轻叹了一口气，转而望向其他人，道：
<br/>　　“你们几个，先把尹庄主和姜长老的尸体抬回去，择日下葬！”
<br/>　　“至于其他人，清理一下战场！”
<br/>　　“是！”
<br/>　　这里整个宗门，几乎都是楚飞烟一手缔造，所以她在众弟子中的威望极高，众人当即领命，开始纷纷行动起来。
<br/>　　……
<br/>　　黑森林最深处，一条小河旁，
<br/>　　叶尘盘腿坐在河边，双眼紧闭，身体一动不动，但是周围又没有任何灵气波动，显然并没有在修炼。
<br/>　　“别藏着了！出来吧！”
<br/>　　叶尘忽然睁开双眼，缓缓开口道。
<br/>　　“嘻嘻！”
<br/>　　一道动听的欢笑声响起，从后方一颗参天大树后面，忽然走出一道倩影，赫然正是萧若曦。
<br/>　　“喂！叶尘，你一个人坐在这里干嘛呢？”
<br/>　　萧若曦双手背在身后，蹦蹦跳跳的来到叶尘跟前，笑嘻嘻的开口道。
<br/>　　叶尘瞥了一按萧若曦，淡淡道：
<br/>　　“你来做什么？”
<br/>　　萧若曦听到这话，顿时嘟着樱桃小嘴，精致完美的脸庞之上，露出一抹黯然之色，
<br/>　　“人家看你心情不好，原本特意过来安慰你的，没想到你还这么凶！”
<br/>　　叶尘摇了摇头，语气依旧淡漠，
<br/>　　“我不需要安慰！你走吧！”
<br/>　　萧若曦听到这话，顿时气得不行，不但没有离开，反而挨着叶尘坐了下来，道：
<br/>　　“胡说！就算再强大的人，也肯定都有脆弱的一面，如果你心里觉得难受，那就要找一个人来倾诉！否则，如果一直憋在心里，是会憋出病来的！”
<br/>　　“我说了，我不需要！”
<br/>　　叶尘再次回绝，然后便打算起身离开。
<br/>　　不料，萧若曦的语气，忽然发生了细微的变化，
<br/>　　“叶尘，当你身在高处，却发现自己总是孤独，无人同你说话，那你身在高处，又意欲何为？”
<br/>　　轰！
<br/>　　听到这话，叶尘顿时全身一震，继而眼眸之中露出一抹浓浓的惊喜之色，同时脱口而出，
<br/>　　“曦月！”
