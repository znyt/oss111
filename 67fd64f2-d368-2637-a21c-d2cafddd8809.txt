　　第388章
<br/>　　本来钟伟虎是不太愿意见程大雷的，这个魔头在他心里留下的印象实在不好。而程大雷也不愿意见到钟伟虎，因为对方不够格，以往这种事情都由和珅负责。不过今天和珅在处理存粮的事情，只能由程大雷屈尊降贵，和钟伟虎当面谈谈。
<br/>　　程大雷到的时候，徐神机正陪着钟伟虎扯淡。徐神机咧开大嘴，唬得钟伟虎一愣一愣的。见到程大雷进来，钟伟虎忙站起来。
<br/>　　“牛将军。”
<br/>　　咦，怎么对自己的称呼又变了。
<br/>　　程大雷已经来到这里三年，钟伟虎也不再装糊涂，以前都称呼他为『程当家』，今天怎么突然改变称呼。
<br/>　　“来了，坐坐。”程大雷在帅岸后坐下：“今天过来是什么事，不到交生活费的日子吧。”
<br/>　　钟伟虎有些无语，薛婷和一干凉州的公子哥被扣押在琴川关当人质，每月还固定要向程大雷缴纳一笔生活费。一想到这个，钟伟虎就像吃了苍蝇一样恶心。
<br/>　　当然，今天是为了重要的事情来的，钟伟虎暂时没提这个。
<br/>　　“长安的事情，牛将军听说了么？”
<br/>　　“呃……”程大雷睁大眼睛，眼珠转了一圈，仔细问：“长安城出了什么事，我这里消息不通，长安的消息传到我这里，至少要好几个月呀。”
<br/>　　钟伟虎不知道程大雷说的是真话还是假话，他顿了顿道：“不瞒牛将军，正义教的匪类造反，如今已经打到长安城下。明帝号召天下诸侯勤王，您这官职是明帝亲口封的，可不能忘了明帝的知遇之恩呐。”
<br/>　　程大雷愣了好大一会，与徐神机对视一眼，对方也是满脸错愕。二人表情流露出的是同一个意思：
<br/>　　这货该不会是没睡醒么！
<br/>　　怪不得一来就对程大雷改变称呼，原来现在是用到程大雷了，所以程大雷不再是匪，而变成了官。
<br/>　　可程大雷这山贼的身份，是板上钉钉的事情，难道你唤我一声牛将军，我就派人去送死么。
<br/>　　“哈，这个么。”程大雷打了个哈哈，道：“明帝与我有知遇之恩，恩同再造，我怎么能忘记，不瞒钟大人说，我现在恨不得背升双翅，飞到长安杀贼。”
<br/>　　“那这太好没有了，不知阁下何时动身。”
<br/>　　“唉……我还没说话呢，你太着急了。”程大雷道：“三军未动，粮草先行，我们琴川关小家小户，老实说，马上就要断粮了，还发愁怎么度过这个冬天呢。不如，凉州城先借我一些粮食，以做军饷。”
<br/>　　钟伟虎睁大眼睛看着程大雷，一副『快别扯淡了』的表情。要是粮食送到你能出兵，我就直接跟你姓。
<br/>　　“假若没有粮食呢？”钟伟虎问。
<br/>　　“没有粮食也要出兵，既然是帝国的将军，就要替帝国杀贼。我们的手下死光了又如何，我自己死了又如何，只要能为明帝分忧，便是死而无憾。”
<br/>　　“牛将军何时出兵？”
<br/>　　“明年吧，明年就可以把粮食凑齐了。”程大雷抿了口茶。
<br/>　　噗！
<br/>　　徐神机险些笑出声来。
<br/>　　钟伟虎脸上神色变幻，显然是十分难堪。
<br/>　　程大雷把茶杯放下，收起脸上的笑容，道：“仔细说吧，今天过来究竟是为了什么事，你如果不把我当傻瓜，我就不会把你当傻瓜。”
<br/>　　用脚趾头想也知道，程大雷不会为了明帝出兵。可宋伯康既然知道，何必要钟伟虎白费力气，那么很显然，他们的目的不是为此。
<br/>　　“程当家果然高明，便也知道瞒不过你。”钟伟虎也正色起来。
<br/>　　“好啦，拍马屁的话就不用说了，我要是想听，在这里随时都能听见。”程大雷冷冰冰道。
<br/>　　“程当家可以不出兵，是否能让我将一人带走？”
<br/>　　“谁？”程大雷再次端起茶杯。
<br/>　　“帝国六皇子，李行哉。”钟伟虎一字一顿。
<br/>　　程大雷一怔，端起的茶杯又缓缓放下，脑海里快速思索着。
<br/>　　是呐，程大雷没有道理拦阻林少羽为父亲报仇，同样，也不该阻拦李行哉去长安替明帝解围。那毕竟是他的生身父亲。
<br/>　　“你可以走了。”程大雷缓缓道。
<br/>　　“那六皇子？”
<br/>　　“帝国六皇子做什么，我想你是没权力过问的。”
<br/>　　钟伟虎还想开口，忽见程大雷板起脸来，重重一拍桌子。
<br/>　　“来人，送客。”
<br/>　　……
<br/>　　赶走钟伟虎后，程大雷亲自去见李行哉。李行哉、福德勒、李婉儿是住在一个院子里，院子里有走廊有花厅，对于李行哉这位帝国皇子，程大雷还是保持着足够的尊重。
<br/>　　程大雷去的时候，李行哉正和福德勒一块儿下棋。李行哉这个人有些奇怪，琴棋书画，诗词歌赋就没有他不懂的，但懂是懂，每样东西也都谈不上精通。
<br/>　　他也养花，他也做画，看到美女他眼睛也直，但这些东西，很难说是他的爱好。他对什么都感兴趣，可对任何事物好像也不感兴趣。
<br/>　　怪不得在长安城，大家都说他是个废物。
<br/>　　程大雷立在李行哉身后看了一会儿，围棋确实也看不懂，李行哉忽地抬头，笑了笑道：“你来啦。”
<br/>　　“嗯，来看看你。”程大雷还没想好怎么说明来意。
<br/>　　“你不来，我也要去找你。”李行哉将面前的棋盘推倒：“我要走了。”
<br/>　　程大雷一怔：“你已经知道了。”
<br/>　　“哈，你真是小看我了，如果我在这里什么都不知道，岂不真成了废物。”
<br/>　　程大雷沉默，的确，长安城里李行哉是公认的废物，但同时，他也是明帝最看重的帝国六皇子，这样一个人，怎么可能什么都不知道。
<br/>　　程大雷沉默片刻，将身上的佩剑摘下，双手递到李行哉面前。
<br/>　　“物归原主。”
<br/>　　李行哉笑了笑：“这不是我的剑，当初在青牛山，我已把这剑当做赎金给你。”
<br/>　　“一场玩笑，岂能当真。”
<br/>　　“这不是我的剑，也不该是我的剑。”李行哉脸上的笑容缓缓收敛：“一位皇子，不该用匹夫剑。”
<br/>　　他负手在窗前，任庭前微风拂面。
<br/>　　“我的剑该以精兵为锋，良马为锷，绕以大河，带以山川，此剑上决浮云，下绝地纪。此剑一用，匡诸侯，诛逆贼，平天下。”
<br/>　　李行哉慢悠悠说着：
<br/>　　“那该是一柄天子剑。”
<br/>　　
<br/>(本章完)
