　　第367章
<br/>　　程大雷顿了顿，端起桌上的茶抿了一口，没有看钟伟虎。
<br/>　　钟伟虎呵呵笑道：“此獠杀人无算，除了程当家谁也擒不住他。只不过他毕竟是在凉州做事，手段残酷，已经惊动了朝廷。程当家能不能将他交给宋大人处理……”
<br/>　　程大雷恍然明白了，无非是上层抢功那一套，当然，程大雷也无所谓什么功劳，不过这崔弄海，程大雷是打算当众杀了，提升山寨凝聚力的。
<br/>　　可这一点，对程大雷来说并不算重要，他毕竟是个看重利益的人。
<br/>　　“这怎么可能，我抓住他可不容易！”程大雷严肃道。
<br/>　　“当然，宋大人绝不会让程当家白辛苦，还要上奏朝廷，为程当家请功。”
<br/>　　程大雷摆摆手：“我不是贪功之人。”
<br/>　　“明白，明白，宋大人有一笔钱粮，是要送给程当家的。”
<br/>　　说着，钟伟虎从身上摸出一张单子，程大雷接过来看了一眼，瞬间睁大了眼睛。
<br/>　　为了换走崔弄海，宋伯康出手可是真够大方的，这笔钱粮若是接过来，足够琴川关三两个月开销。
<br/>　　程大雷按捺住激动，把单子递给和珅，这种谈价钱的事情，还是和珅来做比较合适些。
<br/>　　和珅扫了一眼，道：“想要抓住他，我们可付出了不小的代价，我们大当家也险些遇到危险。这点东西是绝对不够的，除非……加一倍。”
<br/>　　“好，我替宋大人答应了。”钟伟虎站起身道：“现在能不能让我把人接走，答应给程当家的东西，改日就会送来。”
<br/>　　钟伟虎答应得如此痛快，就连和珅都愣了愣，他目光看向程大雷，征求程大雷的意见。
<br/>　　程大雷顿了顿：“我看还是先把东西送过来，再让你们把人接走，你们什么时候能把东西准备好？”
<br/>　　钟伟虎也知道琴川关的脾性，想要占他便宜是万万不能的。他道：“明日，最晚黄昏前。程当家可要说话算话，等东西运到，就让我把人接走。”
<br/>　　“什么话，本大家什么时候说话不算数。”程大雷起身送客：“那咱们就一言为定，明天东西到了，人你可以接走。”
<br/>　　“程当家记得，人可一定要活的。”
<br/>　　“放心啦，放心啦。”
<br/>　　送走钟伟虎后，程大雷显得很激动，没想到原来这个崔弄海这么值钱啊。
<br/>　　“大当家，不把他交给我处置了？”吴用问。
<br/>　　“不用了。还是换钱要紧，把他拎出来，清洗一下，也不要搞得太难看。”程大雷。
<br/>　　“那好吧。”吴用显得颇为遗憾。
<br/>　　程大雷亲自监督这件事，把崔弄海从水牢中吊出来，搁在院子里，几盆清水浇上去，冲净他身上的污垢。
<br/>　　崔弄海被浇得落汤鸡相仿，手筋脚筋被挑断的他，根本没有力气站起来。
<br/>　　“好啦，你走远了，有人要把你接走，你不会死在我这里。”程大雷道：“当然，你也只是多受点折腾而已，最后都活不下去。”
<br/>　　崔弄海抬起头，冲程大雷桀桀笑着，不知怎地，这笑一被程大雷看见，就觉得浑身上下不舒服。明明他已经如此狼狈，可输得却好像自己。
<br/>　　变态果然是变态，不能以常理度之。如果自己知道了变态的想法，自己岂不是也成了变态。
<br/>　　这时候李行哉冲外面回来，道：“程大雷，搞什么呢，我听说你新抓了个狠人。”
<br/>　　李行哉和商队一起去草原，这几日都不在山寨，他刚回来，就来找程大雷，估计是路上又听到什么见闻，来找程大雷炫耀。
<br/>　　“什么狠人，也就一般般啦，喏，他不是在这里么。”程大雷的语气里不无得意，毕竟这崔弄海当真不弱。
<br/>　　李行哉一见就吓了一跳，下意识退到程大雷身后：“你，是你……”
<br/>　　崔弄海抬起头，露出一口森白的牙齿。
<br/>　　程大雷还是第一次见到李行哉如此恐惧，他一张脸竟凝重起来，变得有些苍白。
<br/>　　“你认识他？”这倒是大出程大雷预料，果然呐，变态都是和变态相识。
<br/>　　“怎么可能不认识呢。”李行哉道：“我劝你最好杀了他，越快越好。”
<br/>　　程大雷见李行哉说得如此郑重，倒不知是怎么会事。他重新将崔弄海关进牢里，和李行哉并肩出了院子。
<br/>　　“你不是也在长安待过么，怎么会没有听过他的名字？”二人边走边聊，李行哉道：“大概你去的时候，他已经不在了么。”
<br/>　　“他在长安很有名？”程大雷。
<br/>　　“何止是有名呐，简直是可怕。”
<br/>　　“怎样的人，能让你这个帝国六王子说声可怕？”
<br/>　　李行哉看着程大雷，一字一顿：“他是相府的人。”
<br/>　　听李行哉说起崔弄海的来历，程大雷也觉得不寒而栗。这崔弄海不仅是相府的人，甚至可能是丞相大人的亲生儿子。
<br/>　　关于这一点，长安的传言很多。有人说这是崔相和一个丫鬟所生，有人说这是崔相扒灰了自己儿媳妇，还有更加扯的，也就不足信了。只不过大家唯一可以确认的事情是：这个人生下来就有些不正常的。
<br/>　　十一岁就将贴身丫鬟玩弄至死，大家一开始只以为他是好色，当然，这种事在大户人家也算不得什么。但到后来，大家发现，崔弄海不是好色，而是好杀。
<br/>　　他曾经把刚生育的母马肚子扒开，然后再将小马填进去。他曾强迫一个丫鬟和一个猎犬**，只为看能不能生出孩子，也曾经活活把一个人折磨致死，只为看看一个人体内有多少血……
<br/>　　他做过的事情，莫说让程大雷看见，便是听一听就想要呕吐了。
<br/>　　但不得不说，崔弄海有些学者向前探索的好奇精神。可与其说他是个向真理迈步的学者，倒不如说他是个妖孽。
<br/>　　天生妖孽。
<br/>　　想到这里，程大雷忽然一激灵，怪不得凉州城会不惜一切代价把崔弄海换走，这并不是为了送到朝廷去请功，而是要送到相府做人情。
<br/>　　甚至，很早凉州城就知道崔弄海的存在，毕竟，这样的杀人手法是极其罕见的。
<br/>　　意识到这一点，程大雷忍不住昂头大笑。
<br/>(本章完)
