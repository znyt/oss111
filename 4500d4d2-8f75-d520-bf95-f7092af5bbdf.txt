　　第68章
<br/>　　“面……包？面要包什么？”
<br/>　　“就是一种方的，当然也可以是圆的，面粉发酵的，加盐，烤出来的。”程大雷双手比划着说出来。
<br/>　　“那是……炊饼？”
<br/>　　“不是炊饼，要比炊饼大，也比炊饼软。”
<br/>　　“馒头？”
<br/>　　“比馒头更大，是烤出来的。”
<br/>　　孙九露出惭愧的表情：“大当家见多识广，这些我是没有听说过的。”
<br/>　　程大雷对面包的了解，也就仅限于此，他也并不是烘焙师。他只是看过一部纪录片，了解一个大概过程，但让他做，是做不出来的。所以要参考孙九的意见，给出一个大概思路，二人一同摸索。
<br/>　　“关键……应该在炉子上，我们要砌个炉子。”
<br/>　　程大雷和孙九走出大厅，找来几个有泥瓦匠经验的人，琢磨着面包炉该如何建。
<br/>　　所谓他山之石可以攻玉，帮助苏樱经营酒楼，程大雷还是有几分把握的。纵然他不是厨师，可知道前生很多菜的大概做法，和孙九一起摸索，大多都是能出来的。
<br/>　　毕竟，孙九这个优秀等级的厨师不是白给的，只要一切步入正轨，源源不断的新菜单涌入苏樱的酒楼，想要完成系统任务，并不是什么困难的事情。
<br/>　　他打算培养出一批厨师，送到苏樱那里，这件事自然要交给孙九去做。
<br/>　　“大当家，您实在是太看得起我了，我一定不负所托。”孙九明显有些激动，他没想到这个从前被人呼来喝去的厨子，竟然也有管理别人的时候。
<br/>　　“好好干……”程大雷不明白孙九这感激涕零的表情从何而起，他想着自己是不是也有讲一些领导艺术什么的，不过憋了半天只憋出一句：“我看好你喔！”
<br/>　　制作面包的道路并不太顺利，有时烤糊了，有时烤得夹生，但终究不是太困难的事情。在经历一些失败后，终于成功烤出第一炉面包。
<br/>　　那一天，整个蛤蟆寨的人都尝到了这种叫做面包的食物。这种之前从未见过的食物，别致的味道，在众人间大受欢迎，尤其是在孩子中间。
<br/>　　之前程大雷还担心面包接受度不高，见到这一幕才放下心来，接下来自己只要培养出几个会烤面包的人送到苏樱处就可以了。
<br/>　　只是生产力有限，如今每天制作的面包还没办法让整个山寨的人吃上。如此一来，孙九在蛤蟆寨的身份也是水涨船高，这种感觉可是以前从未体验过的，他这几天走路都觉得飘飘然。
<br/>　　而在成功制作出面包后，程大雷又把孙九找过来，道：“接下来，咱们该研究研究锅底怎么做了。”
<br/>　　……
<br/>　　落叶城。
<br/>　　所谓有钱能使鬼推磨，世间事只要有钱开路，大多都是好行的。
<br/>　　程大雷送来一百两银子，足够苏樱将酒楼重新装潢，也不过是用了十几日时间，里里外外便焕然一新。眼瞅着便到重新开业的日子，苏樱却遇到一件麻烦。
<br/>　　“真的雇不到人么？”苏樱问酒楼掌柜：“工钱不是问题。”
<br/>　　“现在已经不是工钱的问题，我就算拿钱去请，也雇不到人，没有人愿意来咱们酒楼。”
<br/>　　这几天小二和厨子不知怎么会事，突然不辞而别，眼看酒楼就要重新开业，却找不到能用的人。
<br/>　　“还有一件事……”掌柜吞吞吐吐。
<br/>　　“什么事？”
<br/>　　“对不住老板娘，我家里老母病重，怕是不能再在您身前伺候了。”
<br/>　　“你也要走！”苏樱吃惊地道了一声，忽然缓过神来，慢悠悠道：“掌柜的，这里面是有别的事吧？”
<br/>　　“老板娘，是我对不住您，工钱我就不要了。”
<br/>　　“掌柜的，您也是苏家老人了，现在您要找我不留您，多少您给我一句明白话，工钱照结。”
<br/>　　掌柜脸上表情变幻，忍了半天才指了指门外：“里面究竟什么事，老板娘您自己不明白么，何必逼我呢。”
<br/>　　顺着掌柜手指的方向看去，正是生意兴盛的四海楼。四海楼掌柜李大头站在门口，颇有几分意气风发的样子。
<br/>　　那一瞬，苏樱全明白了。李氏——也就是苏家如今哪位夫人，向来视自己为眼中钉肉中刺，这种事苏樱也不是不知情，可没想到，自己已经被赶出苏家，在这破酒楼落脚，李氏还不肯罢休，非要赶尽杀绝才肯。
<br/>　　“需要这么狠么！”
<br/>　　此时日落西山，黑暗降下来，苏樱只感觉四面八方的秋寒涌过来。如果这件事真是李氏从中作梗，事情怕还真不好办。
<br/>　　明天酒楼就要开业，鸡鸭鱼肉也准备不少，可现在酒楼剩下的，只有自己和小蝶，还有廖甲廖乙两兄弟。
<br/>　　“小姐，不若我拿着钱，再去人市上撞撞运气。”
<br/>　　这时代也要专门的人才市场，各行各业大多能从哪里找到，想要雇人要通过专门的经济，也就是这时代所谓的牙行。车船店脚牙，无罪也该杀里的牙行，可如果是李氏打过招呼，怕自己拿钱也找不到人。
<br/>　　“去也是白去，现在不会有人敢应我们的差。”
<br/>　　“小姐，那我们该怎么办？”
<br/>　　苏樱也没有办法，她和小蝶加起来不过两个弱女子，李氏如果有心针对自己，她还真吃不开。或许一走了之，是最后的办法，可她并不想如李氏的意离开落叶城，她心里还是有些不服气的。
<br/>　　正这个时候，一个大脑袋探进来，问：“是苏家小姐么？”
<br/>　　“你是？”苏樱面露困惑，现在称呼她为小姐的人可已经很少了。
<br/>　　“小子孙九，给大嫂见礼。”说着，孙九冲身后挥挥手：“都进来吧，正是这里。”
<br/>　　刚开始苏樱没明白怎么会事，可一听这称呼就全懂了，又是他派来的。
<br/>　　说话光景，十几个随着孙九走进来，既有男人，也有三四十岁左右的妇人。
<br/>　　“大嫂好！”这伙人在苏樱面前站成一排，同时称呼。
<br/>　　苏樱以手扶额，莫名觉得有些头疼。
<br/>　　“住了，住了吧，以后在这里不要这么称呼。”
<br/>　　“大嫂……老板娘，是大哥让我们来的，以后有什么事，您招呼。”
<br/>　　
<br/>(本章完)
