　　第328章
<br/>　　关鱼、秦蛮、赵子龙、张肥、高飞豹五人，皆有百夫不当之勇。
<br/>　　将为兵之胆，见到老大如此拼命，手下也都不含糊。人人士气高涨，跟随着老大冲锋。
<br/>　　十八家头领的人根本不堪一击，他们转过身，把兵器抛下，没命逃跑。
<br/>　　将后背卖给敌人，几乎就注定了他们的失败。
<br/>　　兵败如山倒，逃兵不如狗。
<br/>　　而他们，也根本不是关鱼五人的目标，莫说一万人就是一万头猪，挨个杀也得杀些时间。可只要击杀莫鸣米，一万人也就只剩引颈就戳的份儿了。
<br/>　　关鱼五人个个不甘落后，他们五个暗中也都较着劲儿，包括他们手下的鱼字队，龙字队，蛮字队，豹字队，飞字队也都明白，谁若能杀了莫鸣米，定然是大功一件，在山寨里就可以扬名露脸。
<br/>　　毕竟，平日他们都有竞争，更何况是今日这情况。
<br/>　　于是人人奋不顾身，直冲对方大本营。
<br/>　　莫鸣米没有想到，战场之上的胜负转变来得这样突然。本来以为今日可以凭着重甲军一举击溃蛤蟆寨。可到后来，被击溃的竟然是自己。
<br/>　　从他站立的角度看过去，只见厮杀声不绝于耳，战旗东倒西歪，士兵丢盔弃甲。而蛤蟆寨的气势却如狼似虎。
<br/>　　他们只是一群山贼，他们只是一群山贼！
<br/>　　莫鸣米如此安慰自己，自己的人有一万，加上十八家首领的人，足有两万余人。
<br/>　　可却在一千山贼面前，被打得如丧家犬。
<br/>　　凭什么！
<br/>　　莫鸣米却不知道，蛤蟆寨最初的一百人，都是随程大雷从青牛山打过来的老弟兄。
<br/>　　当初面对漫山遍野的戎族铁骑，大家都撑了下来，何况是现在。
<br/>　　这一百人变成一千人，但胜利的种子已经在每个人心底种下。
<br/>　　莫鸣米再强，比戎族铁骑如何，所以在他们眼里，莫鸣米才是乌合之众。
<br/>　　趟过尸山血海的不是佛祖，便是魔头，这千余魔头，直扑莫鸣米而来。
<br/>　　而今天，却再也没有重甲兵为莫鸣米护阵了。
<br/>　　“将军，撤吧！”莫鸣米的一名亲卫大喊。
<br/>　　咔嚓！
<br/>　　莫鸣米一刀砍下对方脑袋，人头滚滚落在地上。
<br/>　　“再有言退者，与此人同命！”
<br/>　　莫鸣米抬起头，看着关鱼等人已冲杀过来，他咬紧牙关，口中恶狠狠道：“无非是一千山贼而已，惧他作甚，儿郎们，随我杀敌！”
<br/>　　莫鸣米也有一帮死忠亲卫，这都是当初随着他打江山时的手下，眼见今天老大要拼命，他们也顾不得太多。
<br/>　　各自握紧兵器，口中大叫一声，向着蛤蟆寨山贼发起逆向冲锋。
<br/>　　就如同蚍蜉撼树，螳臂当车一般，这些冲锋的浪潮宛若碰到一柄长矛，被硬生生捅破一条道路。
<br/>　　关鱼是何等人也，经过这些日子的历练，早已晋升为顶级武将。如张肥、赵子龙、秦蛮、高飞豹四个也都不是弱手。
<br/>　　莫鸣米手下虽然多，却都不是他们一合之将，只要大刀铁枪蛇矛扬起，就是人仰马翻的结局。
<br/>　　莫鸣米看到这一幕，顿觉胆战心惊，只知道程大雷不好惹，但没想到他手下还有如此多的悍将。
<br/>　　如果自己和他们碰上，怕也免不了被劈为两段的下场。
<br/>　　他打了个哆嗦，一咬牙，口中高呼一声：“撤。”
<br/>　　“大哥，你先撤，我们挡住他。”
<br/>　　喊出这声音的是莫鸣米当年做山贼时的弟兄，到现在，他情急之下，把当初的称呼也喊了出来。
<br/>　　莫鸣米眼睁睁看着对方喊出一声后，向关鱼冲去，紧跟着就被一刀砍掉了脑袋。
<br/>　　莫鸣米眼含热泪，可如今大势已去，自己已无力回天，只能在亲卫的护送下狼狈逃命。
<br/>　　一波波人向关鱼冲去，这阻拦了他们追击的速度。
<br/>　　莫鸣米逃亡的方向是琴川关，只要回到琴川关，坐拥险地，程大雷就绝对攻不进来。
<br/>　　“弟兄们，大家莫慌，等我回到琴川关，重整旗鼓，再为弟兄们报仇。”
<br/>　　莫鸣米回头看着仍跟着自己的士兵，如今残留的不过一千余人，想想出征时的队伍，莫鸣米自己都垂头丧气。可现在，他也只能用这些话安慰弟兄。
<br/>　　“姓莫的，你哪里去！”
<br/>　　忽然一声高喝，只见一女将座下胭脂马，手中绣绒刀，带领三五百手下，向这边冲杀出来。
<br/>　　怎么忘了这娘们，因为落玉寨距离此地颇远，莫鸣米没有专门腾出人手对付他们。
<br/>　　当然，也是因为轻敌的缘故，觉得不至于如此做。若给莫鸣米一个重新再来的机会，他定然先破落玉寨，再用樊梨花的命要挟程大雷。
<br/>　　当然，能不能有下次机会，且要先看这次能否逃命。
<br/>　　若论战斗力来说，落玉寨的战士是不如蛤蟆寨的，也未必是莫鸣米的对手。然而，此刻就连莫鸣米也无战心，更何况是他的手下。
<br/>　　樊梨花冲杀一阵，见留不下莫鸣米，也就撤了。
<br/>　　话说在关鱼带人冲锋的时候，程大雷带着云中龙三家的七百个弟兄保护山寨。
<br/>　　那两千被冻僵的铁甲军，自然只有引颈就戳的份儿。他们有些直接被冻僵了，有些是被程大雷的人手不顺自己杀死了。最后留下的俘虏大概有一千五百余人。
<br/>　　至于十八家首领的人，包括莫鸣米的人，一看事儿不对，直接就逃命去了。
<br/>　　等关鱼带着人回来时，程大雷忙问：“杀了莫鸣米没有？”
<br/>　　“嗨，让他跑了。”关鱼遗憾道。
<br/>　　程大雷也叹口气，如果能够抓住或者直接杀了莫鸣米，那么琴川关将不攻自破，然而现在让他逃了，自己以后还得费点劲。
<br/>　　程大雷开始命令手下清理战场，清点这次的战斗收获。
<br/>　　莫鸣米大营里的人逃得干干净净，物资却都留了下来。缴获的兵器盔甲就不用提了，如投石机、攻城车这样的攻城器械也有不少。至于马匹、粮草、弓箭、帐篷等等需要很长时间才能清点出数目。
<br/>　　在程大雷看来，这次最重要的收获，还是抓到的那一千五百名重甲兵。
<br/>　　这些人都被冻伤了，摘掉盔甲后，人人冻得脸色发青，瑟瑟而抖。
<br/>　　可程大雷也看得出来，这些都强壮如牛，不强壮也撑不起铁甲。
<br/>　　这可都是人才呐。
<br/>
