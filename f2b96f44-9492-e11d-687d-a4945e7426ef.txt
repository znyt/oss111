　　第47章
<br/>　　“这是黄家的帖子，黄员外今年49岁，元配夫人前年才死掉，小樱去了就是正房。”
<br/>　　“可这样是不是有些委屈小樱，小樱才17岁？”
<br/>　　“有什么好委屈的，她没过门就克得夫家灭门，又被山贼糟蹋过，黄员外肯要她就不错了。”
<br/>　　“黄员外家里良田千亩，长子又在守备营任职，怕还是我们高攀。”
<br/>　　落叶城，苏家。
<br/>　　从一间大屋中传出这样的对话。小蝶在门口听了一会儿，悄悄向后院跑去，最后她进入一座破落的院子。
<br/>　　院子内，苏樱正用小米喂几只信鸽。
<br/>　　“小姐，你怎么能做这些下人差事。”小蝶抢过去道：“小姐，你知道不知道，夫人要把你许人了，要许给一个姓黄的老头，胡子都白了。”
<br/>　　“倒是听到一些。”苏樱平静道。
<br/>　　“可是……小姐你怎么能嫁给他，他都可以做小蝶爷爷了。”小蝶诧异苏樱的平静。
<br/>　　“我身上的经历能嫁得出去，便是不错了，何况这次过去还是正房。”
<br/>　　小蝶愣了愣，不知该说些什么。事实上苏樱从蛤蟆寨回来后，状况不是很好，她原来的屋子被二小姐占了，只能被赶到这样一个苏家破败的院子，每月的吃穿用度也缩减许多。
<br/>　　“可小姐心里还是会不开心吧？”
<br/>　　“当然。”
<br/>　　苏樱把手中最后一把小米撒尽。
<br/>　　就在这个时候，落叶城的城门前站着两个人。
<br/>　　『有恶贼程大雷，出身青牛山蛤蟆岭蛤蟆寨，绰号蛤蟆大王，方口阔面，颌下黑须，此贼杀人如麻，十恶不赦，有将其缉拿归案的，赏金百五十两』
<br/>　　还未进城，程大雷先在城门口看到悬赏自己的榜文。
<br/>　　“咦，我的赏金涨了。”程大雷面有得色。
<br/>　　“这上面画得也不像啊，这怎么抓得到人。”徐神机非常遗憾。
<br/>　　“走着，进城！”程大雷牵着马，有些意气风发。
<br/>　　事实上，二人还没走进城门，就被守门的卫兵拦住了。
<br/>　　“喂，干嘛的，我怎么看你长得像悬赏的恶匪！”
<br/>　　在帝国还强盛时，平民是不可随意迁徙的。路引、公验、牌谍等都算是身份证明，想要进城是需要这些东西的。
<br/>　　那么在皇权没落的今天，什么才是什么证明呢？
<br/>　　程大雷随手一把铜钱塞到卫兵掌心，指了指墙上的通缉令：“有很像吗？”
<br/>　　卫兵掂了掂铜钱，约莫有几十枚，悄悄揣到怀里，表面上却依旧板着一张脸。
<br/>　　“你们两个叫什么名字，从那儿来的？”
<br/>　　“在下……”程大雷心中一动，笑道：“在下姓欧名巴，青牛山下人氏。”
<br/>　　“欧巴？”卫兵皱了皱眉头：“倒是个古怪的姓氏。”
<br/>　　紧跟着他又指了指徐神机：“那他呢，他叫什么名字？”
<br/>　　“他叫欧耶，是我本家叔叔。”程大雷揽住徐神机肩膀：“我们是到城里访友，还望大人行个方便。”
<br/>　　“好啦，进去吧，你们既然是从青牛山来的，就该知道青牛山山贼作乱，到晚上可不太平。”
<br/>　　程大雷哈哈笑笑，和徐神机一起走进落叶城。这是他踏足这个世界后，进入的第一个城市，相比之下，蛤蟆岭只能算是穷山沟。
<br/>　　他是边走边看，观察着落叶城的环境，不得不说，城市绿化做得还是相当不错的，几百年的大树随处可见。
<br/>　　城里的商业环境也比程大雷预想得好上很多，街道两旁有不少商铺，也有不少走街串巷的小贩：卖花郎，胭脂摊，菜贩，担柴的，送水的……。
<br/>　　偶尔有一二人骑马从长街呼啸而过，换来行人艳羡的目光。
<br/>　　这种目光程大雷太熟悉了，前世如果有跑车从街上过，大家也是这么个目光。如此看来，虚荣心这东西，倒是古今都一样。
<br/>　　程大雷虽然也牵着一匹马，但这马太瘦，卖相不好，所以也就没有小妹妹向他抛媚眼。
<br/>　　走在街上，与行人擦肩而过，其实看他们身上的衣服，大概就能判断出他们的身份。
<br/>　　穿麻衣的，不用问就是奴隶之类的贱民，能穿棉衣的就证明家境还不错，要是能穿上绸缎，不用问富N代无疑。这里N大于等于2……
<br/>　　那么，就不能是人家白手起家，做到人上人，成为富一代么？
<br/>　　在这个时代，阶级层次是很难打破的，子承父业者居多，当官的祖祖辈辈就当官，经商的祖祖辈辈经商。那些惊才绝艳，你能一步一步爬到高出的人杰不是没有，只是极其稀少而已。
<br/>　　程大雷还算不错，穿着棉衣牵着马，身后还跟着一老奴样的人。这幅打扮不说大富大贵吧，最起码家里条件也不错……。
<br/>　　中午，二人在一家叫做四海楼的客栈落脚。
<br/>　　“小二，来半斤白酒，两斤牛肉。”程大雷刚坐下就扯了一嗓子，自己也过过大侠的瘾。
<br/>　　“老板，咱们这儿没有牛肉，还有……酒小店有，但什么是白酒？”
<br/>　　“……”程大雷。
<br/>　　“还是我来吧。”徐神机清了清嗓子：“向给我们安排一间上房，我们的马在外面拴着，拉到后院好草好料喂着。吃得嘛，来两壶黄酒，半斤羊肉，半斤猪肉，再给抓条鱼熬汤，要快些。”
<br/>　　“得嘞，您稍等着。”
<br/>　　程大雷听得一愣一愣，原来点菜还有这么多门道。首先牛在古代是很重要的劳动力，很长一段时期，杀牛都是犯法的。而白酒的历史是从元朝才开始，虽然这个时代的历史线有些混乱，但程大雷至少能判断出来是在唐以前，所以白酒还没有出现。
<br/>　　小二不多时就将饭菜送上来，道：“两位慢用，有什么需要的您随时吩咐？”
<br/>　　“等一下。”程大雷在桌子上垒下五枚铜钱。
<br/>　　“客官您有什么吩咐？”小二笑纳了，站在一旁。
<br/>　　“坐下来，别着急，你们店都有什么菜，好好给我聊聊。”
<br/>　　“我还是站着给您聊吧……”小二深吸一口气：“本店有……”
<br/>　　程大雷聚精会神，本以为可以听到一篇原生态的报菜名呢，但其实就那十几道菜，烹饪手法不过是蒸，煮，烧烤，就连炒都很少见。
<br/>　　程大雷心脏砰砰直跳，感觉自己似乎发现了一个巨大的商机。
<br/>　　
<br/>(本章完)
