　　第70章
<br/>　　廖甲是第二天上午回来的，还给程大雷带来一封苏樱的信。
<br/>　　那时，程大雷正琢磨着如何酿酒，这时代目前还没有烧酒，也就是蒸馏酒，只有度数极低的酿造酒。程大雷自然不懂如何酿酒，只大概知道蒸馏酒比酿造酒多了一套蒸馏过程，具体怎样操作，其实他不懂。
<br/>　　不过，这不是什么技术难题，有大概思路，总是能琢磨出来的。
<br/>　　“她那边情况还好嘛？有没有提起我？”
<br/>　　“她给大当家捎来一封信，说让我亲手交给大当家。”
<br/>　　程大雷把信拆开，由苏樱执笔的信，自然不像程大雷那样的大白话。都是由之乎者也，四六八句构成。
<br/>　　信中苏樱极尽客套之能事，说如何如何感谢程大雷，滴水之恩，涌泉相报之类的。
<br/>　　紧跟着话锋一转，提前了柳芷。听闻你们佳偶天成，不胜欢喜，祝你们百年好合，白头偕老，如果有朝一日，能再见柳妹妹，一定当面致意。在其后，就是什么桃之夭夭，宜其家室之类程大雷不查书就搞不明白什么意思的话。
<br/>　　但他感受到了其中淡淡的杀意，读着读着，汗水就从额头渗出来。苏樱怎么知道了柳芷的事，肯定是小蝶出卖了自己。
<br/>　　呵，女人……
<br/>　　廖甲道：“大当家，我还带回来三只信封，老板娘嘱咐，如果您有事可以给她写信，让信鸽带回去。”
<br/>　　程大雷之前就从苏樱处取走几只信鸽，不过信鸽送信，主要是利用信鸽归巢的能力，不是把信鸽放出去，它就会去你想去的地方。如今苏樱搬了家，先前带走的信鸽就没有用处了。
<br/>　　“来，取笔墨来，让我给写封信。”
<br/>　　程大雷挥毫泼墨，笔走龙蛇，写好信后，放飞了信鸽。
<br/>　　接着他拍拍廖甲的肩膀：“辛苦你再回去一趟，把山寨那几只信鸽也带上，记得，那边有什么情况，及时给我通风报信。”
<br/>　　落叶城距离青牛山，大概五十里左右，快马往返的话，一天内可以跑个来回。不过，上次大闹黄家后，程大雷在别人面前露了相，通缉他的画像画得更准了。所以，现在他进不了城，不然的话，他还是很想去见见苏樱的。
<br/>　　“大当家吩咐，我自然没问题，只是……大当家，为什么不直接让我把信捎回去。”
<br/>　　“这……”程大雷脸上的表情僵固住，尴尬了半晌，拍了拍廖甲的肩膀：“你这个问题，还真是，哈哈，还真是……”
<br/>　　目光看着天空信鸽消失的方向，程大雷心里琢磨着，今天便应该是苏樱酒楼开业的日子，也不知状况顺利不顺利。
<br/>　　信鸽腾上高空，在眼眸底消失，穿过皑皑云朵，前方出现一座城郭。
<br/>　　苏樱忙碌了一天，她是在晚上打烊才发现这只飞回的信鸽。
<br/>　　店里有小二，后厨有孙九带领的厨房，湖底捞卖的是火锅，其实不用她这个老板娘忙碌什么。
<br/>　　但今天生意实在好得不像话，酒楼不过五十张桌子，一天之内不知道换了多少台。即便如此，还是有很多客人等着，她按照程大雷信里的吩咐，准备了一些烤好的面包免费赠送，也都起到了作用。
<br/>　　自然也遇到些问题，比如两桌客人因为上菜太慢而吵起来，当然，这样算不得什么，给他们送壶酒，打一些折扣便安抚下来。生意上的事，没有一步登天的天才，总是在具体事情中慢慢锻炼出来。
<br/>　　她从前跟在苏四海身边，也是看得多，做得少，如今却是有了实践的机会。
<br/>　　打烊过后，她回到房间，才将那封信鸽带回来的信拿出来。
<br/>　　哼，倒要看看你在心中如何解释……
<br/>　　解释……他为何要向自己解释，为什么自己心里会冒出这么个想法。
<br/>　　“小姐，那恶人信里写些什么？”小蝶问。
<br/>　　“我不知道，我还没看呢。”
<br/>　　口不由心的答了一声，其实苏樱心里也有些好奇，信中究竟会写些什么。
<br/>　　她将信纸抽出竹筒，慢慢打开，盯着上面的内容，慢慢皱起眉头。
<br/>　　这特么是什么意思！
<br/>　　信上只有两个字：在嘛？
<br/>　　苏樱陷入一种莫名其妙的状态，很难描述她此刻的心情。一共送去三只信鸽，是让他有紧急情况和自己联系的，他只送过来两个字，究竟是搞什么，难道让自己回他一下：在
<br/>　　扑腾腾的响声，屋外一只信鸽落在窗台，小蝶赶紧出去，捉着信鸽进来。
<br/>　　“小姐，他又有信来了。”
<br/>　　这次把信打开，上面还是两个字：睡了？
<br/>　　苏樱彻底崩溃。
<br/>　　紧跟着，又有一只信鸽落在屋外，小蝶把它捉进来，苏樱这次已经有了心理准备。
<br/>　　这次好一些，变了三个字：洗澡了？
<br/>　　三封信搁在一起，苏樱满脸问号，她觉得，不是自己搞错了什么，就一定是程大雷搞错了什么？
<br/>　　“看来，要你亲自走一趟，带几只信鸽过去养。”
<br/>　　……
<br/>　　次日天明，湖底捞前面人满为患。
<br/>　　这一幕成了一道风景线，路过的行人有不知详情的，纳闷的问：“这么多人挤在一起干嘛，这里发钱嘛？”
<br/>　　“呵，你还不知道嘛，昨天湖底捞开门，人都挤满了，今天不早点来，占不到位置啊。”
<br/>　　本来是路过的行人慢慢靠近，等看清楚里面的人，忽然一惊：“朱老，您也在！”
<br/>　　这位『朱老』是落叶城城主的管家，所谓管家并不是如很多人所想的那样，站在门口狐假虎威的欺负一下来访的客人。换在程大雷前生，一个大家族的管家至少也相当于一个大企业总监级别的人物。
<br/>　　所以这朱老，在落叶城也是一个响当当的人物。
<br/>　　“朱老，您什么好吃的没吃过，您还在这里排队。”
<br/>　　“哈！我是替少城主办事，少城主几个朋友从外阜过来，要我在这里占个位置。”
<br/>　　“少城主亲自来这里吃饭！”
<br/>　　“哈，也是听说的。”
<br/>　　苏樱因为昨天程大雷的崩溃三连，一夜没有睡好，今天起床也稍微晚些。等她吩咐人打开，看到外面情景时，也是吓了一跳。
<br/>　　
<br/>(本章完)
