　　轰！
<br/>　　叶尘直接随手一抓，一道巨大的金色巨掌虚影，瞬间便将地上的屠苏元让一般抓了起来。
<br/>　　然而，
<br/>　　就在叶尘准备出手，将屠苏元让灭杀的时候，
<br/>　　轰隆隆！
<br/>　　突然一道响彻天地的声音，包含着无尽怒火，犹如雷音滚滚般，陡然自九天之上传荡开来！
<br/>　　“是哪个天杀的，害死了我的儿子！我屠苏霸天，今日要将他碎尸万段！！！”
<br/>　　听到这道声音响起，周围围观的人群之中，一下子就炸开了锅。
<br/>　　“屠苏霸天！莫非就是屠苏家的那位老家主，云丹宗十大长老之首的那位？”
<br/>　　“废话，除了这位屠苏长老，这个云丹城还有第二个叫屠苏霸天的吗？”
<br/>　　“这位屠苏长老可是真正的化神天君，而且在整个云丹星域，实力都足以挤进前五！”
<br/>　　“这下那小子要倒霉了！”
<br/>　　……
<br/>　　在众人的纷纷议论声中，远处高空之上，出现了一道霸气侧漏的紫袍老者身影。

<br/>　　被叶尘擒住的屠苏元让，看到这个紫袍老者出现，顿时脸色狂喜，当即高声呼喊了起来。

<br/>　　“小子！就是你杀了我的儿子？”
<br/>　　“爷爷，救我！快救我啊！！”
<br/>　　就在屠苏霸天正疑惑不解之时，屠苏元让再次高呼求救起来，屠苏霸天只好将心头的疑惑暂时收了起来，冷声道：
<br/>　　“小子！本座不管你是用了什么卑鄙手段，杀害了我的儿子，现在，放了我的孙儿，本座可以考虑留你一条全尸！否则的话，本座定会让你求生不得，求死不能！”
<br/>　　说出这话的同时，屠苏霸天身上的气息也完全释放了出来。
<br/>　　轰隆隆！
<br/>　　不但下方的天藏阁全部轰然倒塌，方圆数千米之内的所有建筑，尽数夷为平地。
<br/>　　甚至，连整个云丹城，似乎都剧烈晃动了下来，仿佛发生了超大型地震一般！
<br/>　　就连叶尘感受道屠苏霸天身上的气息，也不由得眉头微微一皱，
<br/>　　“化神后期吗？难怪能够成为云丹宗十大长老之首！”
<br/>　　单单论修为而言，这个屠苏霸天的实力，比之前的血天海，还要强上一线。
<br/>　　不过，真正让叶尘感到有些威胁感的，并非是这屠苏霸天，而是他能够隐约感受到，周围远处，似乎有着几道，丝毫不比屠苏霸天弱的气息！
<br/>　　毕竟，这云丹星域，乃是六级星域，纵然因为炼丹师居多的缘故，战斗力并不太强，可却也不是之前的巨门星域，所能相提并论的！
<br/>　　倘若周围按几道气息，一起对他出手，又活在在打斗的时候，突然对他偷袭，恐怕会给他造成不小的麻烦。
<br/>　　所以，对付眼前这个屠苏霸天，他必须以强势手段，速战速决！
<br/>　　唯有如此，才一举震慑住那几道气息，让他们来不及出手支援。
<br/>　　一念及此，叶尘嘴角微微一扬，
<br/>　　“老头，想要你孙子的小命是吧？好，我给你！”
<br/>　　说完这话，叶尘直接将屠苏元让，朝屠苏霸天丢了过去。
<br/>　　看到这一幕，周围众人纷纷点头，
<br/>　　“果不其然啊！面对屠苏长老这样真正的化神天君，这小子也嚣张不起来了吧！”
<br/>　　“识时务者为俊杰，化神天君的手段，可不是我们能想象的！”
<br/>　　“早知如此，又何必当初呢！即便他放了屠苏家的大公子，屠苏长老也不会放过他的！”
<br/>　　……
<br/>　　在众人的议论声中，屠苏元让被叶尘像丢垃圾一般，丢向了屠苏霸天。

<br/>　　眼看屠苏元让飞到跟前，屠苏霸天就要伸手去接，却猛然察觉到一丝异样，不由得脸色微微一变，慌忙倒退了数十米，
<br/>　　“彭！”
<br/>　　就在屠苏霸天倒退的那一瞬间，人在空中的屠苏元让，竟然如同一枚炸弹般，轰然爆炸开来！
<br/>　　原来，
<br/>　　叶尘假意将屠苏元让丢了过去，让对方和众人都误以为，他认怂了，却早已经悄悄，将一道攻击，打入了屠苏元让的身体之中。

<br/>　　屠苏霸天足足愣了好一会，方才反应过来怎么回事，望着那已经化成一团血雾的屠苏元让，顿时气得眼睛都要喷出火来了！
<br/>　　而这时，就在屠苏霸天愣神的功夫，叶尘早已经通过缩地成寸的神通，紧追了上来，

<br/>　　轰！
<br/>　　叶尘就是要趁对方反应不及，但对方一个措手不及，然后速战速决，所以一上来，就直接将逆生百转，开启到了第六重！

<br/>　　在所有人难以置信的目光中，叶尘身上的气息，竟然呈爆炸式的增长，瞬间就翻身了几十倍！

<br/>　　“难道他之前隐藏了实力？”

<br/>　　……

<br/>　　“彭！”
