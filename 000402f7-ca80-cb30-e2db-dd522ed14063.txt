　　“快撤退！所有单位，速速撤退！”
<br/>　　指挥部看到叶尘，转眼就灭掉了十几架战斗机，当即便下达了撤退的命令。
<br/>　　可惜，那些军舰虽然移动速度很快，但是在叶尘的面前，却犹如龟速一般。
<br/>　　“破！”
<br/>　　叶尘身体一晃，便已经追上其中一只军舰，然后扬起手中的紫琼剑，直接一剑斩落下去！
<br/>　　军舰上的米国大兵，只看见天空之上，一道耀眼的紫色光芒，朝他们冲击而来。
<br/>　　砰！
<br/>　　轰隆！！
<br/>　　咔嚓！
<br/>　　下一刻，军舰上的所有米国士兵，只感觉到一股强大的冲击力，直接将他震飞了出去！
<br/>　　而在其他军舰上的士兵看来，那艘近三万吨级的军舰，竟然被那道数百米长的紫色剑芒，生生劈成了两瓣！
<br/>　　“哦！我的上帝！”
<br/>　　“这到底是什么东西？难道是超人？还是外星人？”
<br/>　　“快跑啊！！”
<br/>　　……
<br/>　　整个米国海军阵营中，顿时一片暴乱，所有人在那无比震撼的一幕之下，已经完全放弃了抵抗的念头，恨不得赶紧逃离战场，避开眼前这个可怕的恶魔！
<br/>　　然而，叶尘一剑将一艘大型军舰劈成两半之后，却丝毫没有要停手的意思，纵身一跃，已经又朝其他军舰追了过去。
<br/>　　轰轰轰！
<br/>　　轰隆隆隆隆！
<br/>　　叶尘手持巨剑，竟然犹如菜刀切西瓜一般，每挥出一剑，就斩断一艘军舰。
<br/>　　一转眼的功夫，五艘战列舰，竟然全部毁在了他的手上。
<br/>　　而此时，看到这一幕的米国海军众高层，顿时开始慌了，
<br/>　　“快！用导弹给我轰！给我狠狠的轰！绝对不能让他上岛！”
<br/>　　……
<br/>　　随着这道命令下达，
<br/>　　轰轰轰！
<br/>　　一枚枚导弹，自珍宝港海军基地射出，犹如道道流星陨石般，朝叶尘所在的那一片海域爆射而去！
<br/>　　到底是米国海军在太平洋战区的中心基地，防卫自然不是别处可比。
<br/>　　转眼之间，数十枚导弹转眼而至，足以将叶尘脚下的那片海域生生炸平。
<br/>　　然而，叶尘竟然依旧没有后退半步，反而迎头冲了上去！
<br/>　　“给我，灭！”
<br/>　　眼看那数十枚威力强劲的导弹，已经飞至跟前，叶尘再次扬起紫琼剑，狠狠一剑劈落下去！
<br/>　　咔嚓！
<br/>　　一道长达上千米的紫色剑芒，在夜空之中划过，竟然将天空硬生生撕开了一道长长的裂痕！
<br/>　　下一刻，那数十枚导弹的速度，猛然变慢了无数倍，继而又被吸入那道裂痕之中。
<br/>　　转眼之间，那些导弹，就如同凭空消失了一般。
<br/>　　再然后，叶尘身体一晃，已经横跨数十里，而珍宝港海军基地，也已经是近在眼前！
<br/>　　“什么！”
<br/>　　指挥中心的众高层，顿时一个个全傻眼了，
<br/>　　“这家伙，竟然如此轻易，就突破了我们的防线？”
<br/>　　“难道他是魔鬼不成？”
<br/>　　“一旦让他冲入基地之中，那我们可就彻底完蛋了！”
<br/>　　“中将阁下，我们现在该如何是好？”
<br/>　　所有人纷纷望向太平洋战区的最高长官，米歇尔中将。
<br/>　　米歇尔同样眉头紧皱，略微沉吟了片刻后，一咬牙道：
<br/>　　“快！导弹继续轰炸！同时出动核潜艇群进行拦截！我就不信拦不住他！”
<br/>　　……
<br/>　　踏踏踏！
<br/>　　叶尘化作一道流光，速度已经超过音速，在海面之上留下一道道残影。
<br/>　　眼看就要冲入基地之中，前方陡然又出现了数十枚导弹！
<br/>　　继而，脚下也跟着猛然一震！
<br/>　　叶尘不及多想，当即腾空而起，瞬间跃至上千米的高空！
<br/>　　而几乎同时，数十枚巡航导弹，自海面之下爆射而出。
<br/>　　叶尘顿时被这，一上一下的导弹群，包裹在了中间。
<br/>　　“哼！”
<br/>　　叶尘顿时冷哼一声，
<br/>　　“十方无敌！”
<br/>　　唰唰唰！
<br/>　　漫天的紫色剑芒飞舞，在叶尘周围上下，形成了一个，直径达数百米的剑气护罩。
<br/>　　嘭嘭嘭！
<br/>　　那漫天的导弹，与叶尘的剑气相撞之后，顿时尽皆爆炸。
<br/>　　“打中了？”
<br/>　　海军基地指挥部，米国军方的众高层，全都死死的盯着屏幕上的那一团炮火。
<br/>　　然而很快，所有人失望了，
<br/>　　唰！
<br/>　　那炮火还没有散尽，一道金色人影从里面冲出！
<br/>　　叶尘丝毫无损，而且转眼之间，已经又横跨数十里，此刻已经冲入海军基地之中了。
<br/>　　“噗通！”
<br/>　　总指挥部中，众人全都一屁股坐倒在椅子上，一个个的脸色，比死了爹妈还要难看。
<br/>　　“完了！这下彻底完了！”
<br/>　　米歇尔如同丢了魂了一般，两眼呆滞无神，口中失声喃喃道。
<br/>　　连防护导弹都阻止不了叶尘的步伐，一旦让他杀入基地之中，基地中的那些普通士兵们，又如何是这恶魔的对手？
<br/>　　就在这时，一名白人军官冲了进来，气喘吁吁道：
<br/>　　“中将阁下，那个恶魔已经杀进来了，请您还有在座的诸位，务必赶紧撤离！”
<br/>　　米歇尔微微一愣，旋即叹了一口气道：
<br/>　　“来不及了！以那个家伙的恐怖实力，我们根本没有任何逃脱的可能！”
<br/>　　其余众人的脸色，顿时更加惨然。

<br/>　　“趁着现在还有些时间，速速帮我连线白宫，我要向总统阁下，亲自汇报这里的情况！”

<br/>　　叶尘冲入珍宝岛之中，早已经大开杀戒。

<br/>　　短短十几分钟过后，整个珍宝港海军基地，已经完全沦为了废墟。
<br/>　　“嗯？地底好像还有波动，如此隐蔽的地方，应该就是他们的指挥部吧？”
<br/>　　叶尘感应到下方的情况之后，再次举起手中的紫琼剑，狠狠劈落下去……
